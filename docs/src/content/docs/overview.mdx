Den is an aspect-oriented Nix framework: you write features, not host files. A pure algebraic effects pipeline resolves aspects into NixOS, Darwin, and home-manager configurations automatically. Read more about [why Den exists](/motivation/).

## Understand

### Core Principles

Den's [design philosophy](/explanation/core-principles/): features over hosts, aspects over modules, composition over duplication.

### Aspects

The primary concept in Den. Aspects bundle per-class config into composable, reusable units.
